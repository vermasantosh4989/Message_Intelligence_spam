# Message Intelligence System

## Project Overview
The Message Intelligence System is a machine learning classification project designed to classify messages as Spam or Legitimate using KNN, SVM, and Naive Bayes.

## Objective
- Build a spam detection system
- Compare classification algorithms
- Evaluate model performance using standard metrics

## Models Used
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Gaussian Naive Bayes

## Preprocessing
- Missing Value Handling (SimpleImputer)
- Feature Removal
- StandardScaler
- SMOTE

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Imbalanced-Learn

## Conclusion
This project demonstrates machine learning techniques for spam message classification and compares the performance of KNN, SVM, and Naive Bayes.
